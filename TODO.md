## High

### Physics Refactor Incomplete (Issue #219)

Three known problems remain from a prior physics refactor:

1. **EffectGroup/EffectLists** — Particle effects only update when their parent
   object is painted, so off-screen particles don't animate. Visible as choppiness
   in slow-motion.
2. **Overlays** — Particle effects on overlays only refresh once per tick, causing
   stutter in slow-motion.
3. **Collision detection** — Some collision checks still fire only once per tick
   rather than per physics update cycle.

**Owner**: `gcabbage` filed this. No assignee. Requires physics subsystem knowledge
and a build environment with the DirectX SDK.
